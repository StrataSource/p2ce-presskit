Portal 2: Community Edition Presskit

Hello member of the press and/or interested P2CE fan! This is a collection of files you can use when reporting on P2CE.

Folder structure:

Screenshots: Screenshots of the game with UI and all
Cinematic Shots: Shots of maps running in P2CE with all UI turned off
Logos: Our Logos and Icons, subject to the guidelines at https://branding.portal2communityedition.com
